# Backup-Webseite (GitHub Pages)

Die statische Backup-Seite liegt in einem **eigenen öffentlichen Repo** (GitHub Pages funktioniert im Free-Plan nur bei öffentlichen Repos):

**CookView (Startseite):** https://nordlicht-lab.github.io/rezeptbuch-backup/  
**Rezept-Backup:** https://nordlicht-lab.github.io/rezeptbuch-backup/backup.html  
**Repo:** https://github.com/nordlicht-lab/rezeptbuch-backup

Änderungen an der Webseite dort committen und pushen – GitHub Pages deployt automatisch.

Der Ordner `web/` hier ist eine Kopie zum lokalen Testen:

```bash
open web/index.html
```
