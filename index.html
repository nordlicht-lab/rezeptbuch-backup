Rezept aus dem Browser sichern

Wenn der automatische Import nicht klappt: Rezept-JSON aus der Browser-Konsole kopieren und hier in lesbares Markdown umwandeln – zum Speichern, Bearbeiten oder Einfügen in Word.

1 JSON-LD vom Rezept finden

Viele Rezeptseiten bieten strukturierte Daten im JSON-LD-Format. Um diese zu importieren:

  1. Öffne die Rezeptseite in deinem Browser.
  2. Öffne die Entwicklerkonsole:
    Chrome / Edge: Cmd ⌘ + Option ⌥ + I (macOS) oder F12 (Windows)
    Firefox: Cmd ⌘ + Option ⌥ + I (macOS) oder F12 (Windows)
  3. Wechsle zum Tab Konsole / Console.
  4. Führe folgenden Befehl aus (kopieren mit dem Button):
[...document.querySelectorAll('script[type="application/ld+json"]')]
  .map(s => s.textContent)
  .map(t => { try { return JSON.parse(t) } catch { return t } })

In der Ausgabe den Teil mit "@type": "Recipe" suchen. Mit Rechtsklick → Objekt kopieren (Chrome) bzw. den kompletten { … }-Block markieren und kopieren (Firefox).
